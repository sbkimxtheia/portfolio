# 마인크래프트 외

> 프로그램, 앱, 웹 모두 마인크래프트 서버와의 연동 가능

### 프로그램 개발

- 데스크탑(Windows, macOS) 프로그램
- 모바일(안드로이드, 아이폰) 앱
- 디스코드 봇(마인크래프트 연동 가능)

### 웹

- 간단한 웹사이트 제작

### 기타 사용 가능 언어

Java, Kotlin, C#, Dart(Flutter)
